# Requirements

## プロジェクトゴール
- 夫婦で日常情報を無理なく共有し、毎日自然に開きたくなる生活アプリを作る。
- iPhone ネイティブ風の見た目と、入力負担の少ない体験を重視する。

## 初期開発方針
- まずは **動く最小構成** を優先する。
- docs を先に整備し、仕様変更しやすい状態を作る。
- Expo Go (iOS) で実機確認しながら段階的に進める。

## MVP（最小機能）
### 1) 買い物リスト
- 共有チェックリスト
- チェック済み項目の非表示/完了扱い
- よく使う項目のサジェスト保存

### 2) 見たい映画リスト
- タイトル保存
- サブスク配信先メモ
- 視聴済み状態管理

### 3) LINE 定型文送信（将来連携前提）
- 定型文をワンタップでコピー/送信導線
- 定型文の追加・編集余地を確保

## 非機能要件
- Expo SDK 54 互換
- TypeScript strict モード
- app directory（Expo Router）構成
- Windows 環境でも開発しやすい手順
