# UI Guidelines

## デザイン原則
- 見やすさ最優先（情報密度を上げすぎない）
- 片手操作を意識したレイアウト
- カップルで共有する前提で、柔らかいトーンを採用

## カラーパレット（初期）
- Primary: `#5C6BC0`
- Surface: `#FFFFFF`
- Background: `#F6F7FB`
- Text Primary: `#1D2736`
- Text Secondary: `#455468`

## タイポグラフィ
- 見出し: 700
- 本文: 400
- 最小フォントサイズ: 14

## コンポーネント方針
- カード UI を基本とする
- 余白は 8pt グリッドで統一
- 角丸は 12〜16 を基本値とする
